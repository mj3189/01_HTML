표 관련 기본 태그

table(표) 태그

        -웹 문서에서 자료(데이터) 정리를 위해 사용하는 태그이다. 
        -table 태그는 행, 열을 작성하기 위한 판을 나타낸다.

tr(Table Row)태그 

    -테이블의 한 행을 나타내는 태그(가로)

td(Table Data) 태그

    -한 행의 한 컬럼(열)을 나타내는 태그(세로)
    ==엑셀의 셀 하나

th(table Header,테이블의 머리 표에 적는 내용의 타입을 굵게 쓰는 것) 태그

    -두꺼운 글씨, 가운데 정렬이 자동으로 적용된다.

    -테이블에 대한 설명을 작성하는 태그

테이블 관련 추가 태그2

    [테이블 영역 지정 태그]
    thead 태그: 테이블 상단 영역(컬럼 제목 작성 부분.. 타입)
    tbody 태그: 테이블 중간 영역(내용, 값을 작성하는 부분)
    tfoot 태그: 테이블 하단 영역(합계, 총합, 인원 수 등)

    (*참고 사항)
table 태그 내부를 thead, tbody, tfoot으로 나누지 않으면 모든 tr, td, th 태그는 tbody 내부에 작성된 것으로 취급한다

행, 열 병합 속성 : rowspan, colspan


    (병합이라는 의미보다 몇 칸을 차지할 것인지에 대한 의미로 해석)

rowspan : 행을 몇 칸 차지할 것인지 (행 병합, 세로로 병합됨)
colspan : 열을 몇 칸 차지할 것인지 (열 병합, 가로로 병합됨)
